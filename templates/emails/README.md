# QyberHost Email Templates

This directory is organized by service category so email themes can be dropped in without changing the overall structure.

## Categories

- `account`
- `hosting`
- `servers`
- `domain`
- `ssl`
- `mail-services`
- `security`
- `backup-monitoring`
- `billing`
- `support`
- `marketing`
- `legal`

## Notes

- Existing imported theme files were preserved and moved into matching categories.
- Newly created files are starter placeholders and can be replaced with final themed HTML.
- Shared visual assets are kept in `images/`.
- A reusable starter layout is available in `library/base-email-layout.html`.
